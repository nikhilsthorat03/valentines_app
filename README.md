# Valentine's Day Invitation App

A professional Streamlit web application for a Valentine's Day date invitation with an interactive "No" button that moves away from the cursor.

## Features

- Colorful animated gradient background
- Interactive "No" button that evades the cursor
- Professional popup invitation with date details
- Responsive design with no scrolling

## Installation

```bash
pip install -r requirements.txt
```

## Running the App

```bash
streamlit run valentines_app.py
```

The app will open in your browser at `http://localhost:8501`
